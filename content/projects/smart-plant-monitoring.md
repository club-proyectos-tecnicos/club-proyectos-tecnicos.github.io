## Resumen

**Prediccion de trayectoria en deportes** es un proyecto orientado a estudiar el movimiento en situaciones deportivas y construir una base tecnica para estimar trayectorias a partir de datos.

La propuesta combina analisis de datos, modelado predictivo y visualizacion para entender como evoluciona una trayectoria y que variables pueden ayudar a anticipar su comportamiento.

La memoria completa se mantiene como documento descargable para quien quiera revisar la metodologia, el desarrollo y las conclusiones con mas detalle.

## Objetivo

El objetivo del proyecto es desarrollar una aproximacion tecnica para:

- representar trayectorias deportivas de forma estructurada;
- analizar variables relevantes del movimiento;
- preparar datos para modelos predictivos;
- evaluar resultados y limitaciones;
- documentar el proceso de forma reproducible.

## Enfoque

El proyecto se plantea como una aplicacion practica de analisis tecnico en deporte. La idea central es transformar observaciones de movimiento en datos que puedan estudiarse, compararse y utilizarse para construir modelos.

El trabajo se apoya en un flujo sencillo: preparacion de datos, exploracion, modelado, validacion y comunicacion de resultados. Esta estructura permite separar la parte experimental de la explicacion final y facilita continuar el proyecto en futuras iteraciones.
