## Resumen

Este proyecto investiga cómo factores contextuales y patrones de comportamiento en deportes competitivos pueden ayudar a predecir o clasificar decisiones de lanzamiento bajo presión.

## Qué problema explora

Explorar si la recopilación y el análisis de datos sobre situaciones competitivas puede revelar patrones en la toma de decisiones y si esos patrones son predecibles con modelos de machine learning.

## Qué se está construyendo

Un análisis que incluye:
- Recopilación de datos de situaciones competitivas.
- Identificación de variables relevantes (contexto, presión, historial).
- Desarrollo de modelos predictivos de clasificación.
- Evaluación de precisión y validación cruzada.

## Qué se espera aprender

- Cómo estructurar datos deportivos para análisis.
- Aplicación de técnicas de machine learning a predicción de comportamiento.
- Interpretación de coeficientes de modelos.
- Cómo comunicar hallazgos complejos de manera clara.

## Próximos pasos

- Completar recopilación y limpieza de datos.
- Entrenar y validar modelos predictivos.
- Documentar variables influyentes y limitaciones.
- Preparar visualizaciones y memoria del proyecto.
